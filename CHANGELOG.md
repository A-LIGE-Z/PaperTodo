## 版本记录

按功能阶段整理，版本号只用来分层阅读，不对应公开发布。每个版本尽量只讲一个方向。


### v0.1 起步

- 定下「桌面纸片」的样子：没有主窗口，每张纸都直接待在桌面上。
- 做出第一张能用的待办纸，内容保存在本地。
- 明确不走 WebView / Tauri / Electron，也不上架商店。

### v0.2 纸片能记住自己

- 可以同时摆很多张纸，各自独立。
- 重开程序后，之前没删的纸会自动回到原位。
- 关闭只是收起来，删除才会真的移除。
- 位置、大小、置顶、内容都会自动记住。
- 多存一份备份，主文件坏了能恢复。
- 纸片跑到屏幕外时，启动或显示会自动把它拉回来。

### v0.3 待办用起来

- 点一下勾选完成，再点取消。
- 完成的事项显示删除线并淡化。
- 回车新增一行，在空行上按退格删掉它。
- 底部有个 `＋` 区域，点一下接着写。
- 可以删单条，也可以一键清理所有已完成。
- 始终留一个空行给你输入，连续记事不被打断。

### v0.4 托盘入口

- 加入托盘图标，作为没有主窗口时的统一入口。
- 从托盘就能新建待办 / 笔记、显示或隐藏全部、退出。
- 托盘里列出当前所有纸片，可单独显示或隐藏某一张。
- 左上角的类型图标兼作置顶开关。
- 程序目录放一个 `PaperTodo.ico`，托盘就用它当自定义图标。
- 标题太长会自动省略，菜单不会被撑宽。

### v0.5 顺手与好看

- 右键菜单按场景分好组，找命令更快。
- 滚动条换成和纸片更搭的细样式。
- 删除确认弹窗改成自己画的样式，和整体风格一致。
- 边框、阴影、圆角、悬停效果统一打磨。
- 去掉 Windows 默认的蓝色选框，按钮更干净。
- 待办勾选框换成暖褐圆角加手绘对勾，更有纸的味道。

### v0.6 拖动与粘贴

- 拖动待办排序时，会显示要落在哪一行。
- 右侧的拖动手柄点击区域更大，更好抓。
- 拖动过程顺滑，松手才真正调整顺序。
- 拖动时底部会变成删除区，拖进去就删掉。
- 从别处粘贴多行文字会自动拆成多条，并清掉常见的 `-`、`*`、`1.`、`☑` 等前缀。
- 一次粘贴最多 200 条，避免超大段文字把界面拖卡。

### v0.7 笔记纸与 Markdown

- 新增笔记纸，支持轻量 Markdown。
- 编辑和预览两种状态可以切换。
- 支持标题、引用、列表、代码块、超链接等常用格式。
- 修好了用快捷键排版后撤销 / 重做错乱的问题。

### v0.8 输入更稳

- 修好首行空待办按退格后光标丢失的问题。
- 优化中文输入法下的回车，打拼音时不会误触新增一行。
- 保存失败时通过托盘气泡轻提示，不打断你。
- 修复若干稳定性问题。

### v0.9 更可靠

- 修复多个稳定性问题。
- 解决了同时启动多个实例时偶发的崩溃。

### v0.10 笔记自动排版

- 笔记纸默认就以排版后的样子显示。
- 点进内容即可编辑，点开后自动恢复排版预览。
- 去掉切换预览的快捷键，改成点进 / 点出的自然方式。
- 点标题栏或空白处会自动收起编辑、回到预览。
- 修复点击预览文字偶发的崩溃。

### v0.11 主题与系统设置

- 托盘菜单加入浅色、深色、跟随系统三种主题。
- 深色模式用暗炭背景配暖乳白文字，保留纸的质感。
- 主题选择会记住，下次打开自动恢复。
- 加入开机自启动选项。
- 修好深色模式下输入光标看不见的问题。

### v0.12 菜单更精致

- 右键菜单和托盘菜单用上圆角、更紧凑的行高。
- 精简菜单命令，去掉用不上的系统项。
- 主题切换从竖排菜单改成横向分段选择器。
- 去掉菜单项悬停时的系统蓝框。
- 菜单整体收紧，少挡屏幕。

### v0.13 待办撤销

- 待办支持多步撤销 / 重做，最多回退 100 步。
- 勾选、清理、新增、删除、拖动排序都能撤销。
- 编辑文字时 `Ctrl+Z` 先撤字，撤无可撤再回退到整条操作。
- 右键某条待办会先把光标落到它上面，操作目标更明确。

### v0.14 更轻快

- 优化笔记纸的显示，频繁点击切换时更流畅。
- 清理冗余代码，让程序保持轻量。

### v0.15 数据更安全

- 当主文件和备份都损坏时，宁可不动也不用空白覆盖你的旧数据。
- 保存改到后台进行，不卡界面。
- 万一程序崩溃，会先抢救一份 `data.crash_recovery.json`，且不覆盖主数据。
- 修复若干稳定性问题。

### v0.16 单实例与唤醒

- 已经在运行时再次启动，不会开第二个程序，而是唤醒当前的。
- 双击托盘图标会把纸片显示、置顶并带到最前。
- 修复大量稳定性问题。

### v1.0 正式版

- 启用全新的应用图标，在托盘和任务栏、深浅背景下都更清晰精致。
- 修复大量稳定性问题，整体更流畅。

### v1.1 胶囊模式

- 纸片可以折叠成一颗小胶囊，大幅省桌面空间。
- 折叠和展开做了平滑的过渡动画，过程中内容不会挤变形。
- 修复大量稳定性问题，动画更顺。

### v1.2 多语言

- 界面文字支持中文、英文、日文、韩文，随系统语言显示。

### v1.3 托盘与启动参数

- 托盘里删除纸片更省地方，删除确认就在同一行完成。
- 支持用命令行 / 快捷键控制：显示全部、隐藏全部、切换显隐、新建待办、新建笔记、保存退出，方便配到外部快捷键工具或快捷方式。
- 正常启动时不再多做一次写盘。
- 修复大量稳定性问题，性能和动画更好。

### v1.4 打磨

- 优化性能与动画。

### v1.5 笔记浏览与胶囊贴边

- 笔记纸的编辑和浏览改用同一个文本框切换，浏览时跳动更少。
- 新增「胶囊自动贴边」：折叠的胶囊会自动排到屏幕右上角，半隐藏在边缘，鼠标靠近时滑出来。
- 在托盘里连续删除多张纸时，菜单不再每次自动关掉，整理更顺手。

### v1.6 Markdown 三档与胶囊动画

- 笔记纸保留编辑 / 浏览双模式，编辑时专注写，浏览时看排版。
- 浏览态的标记淡化更聪明，不会误伤普通括号或任务框 `[x]`；无序列表显示成圆点，任务列表保持整齐。
- Markdown 解析新增「不启用 / 启用 / 增强」三档，默认增强。
- 重做了胶囊折叠 / 展开动画，减少透明窗口缩放时的碎帧。
- 胶囊文字过渡更稳，收缩到最后更接近胶囊形状。
- 首次使用时默认开启胶囊模式和自动贴边。

### v2.0rc1 测试版

**边缘胶囊**

- 重构贴边胶囊的激发态：从边缘胶囊打开纸片后，可复用同一套正常胶囊 UI 作为边缘标签，不再维护一套单独重绘的界面。
- 贴边激发态新增实线外描边，深色模式下对比度更清楚；笔记图标和待办删除线在深色主题里也更亮。
- 新增「展开纸片时保留边缘胶囊」设置：开启时保留右侧高亮胶囊并占用原槽位；关闭时展开期间隐藏它并释放槽位，纸片收回胶囊后再显示。
- 修复从边缘胶囊展开 / 收回时的闪烁、错误位置唤出和隐藏后快速点击导致的位置错乱。
- 修复纸片缩回胶囊后，边缘胶囊先硬切到侧边再显示的问题；现在只走一次正常缩回路径。
- 修复边缘胶囊描边激发态拖动排序不稳的问题；外壳、描边和左侧内容区现在共用同一套非关闭区域拖动入口。
- 优化描边激发态边缘胶囊的拖动启动手感，展开保留的高亮胶囊更容易直接上下拖动排序，同时普通贴边胶囊仍保留防误拖阈值。
- 修复描边激发态边缘胶囊拖动排序时，轻微横向位移会过早取消手势的问题；现在横向移动只取消点击激活，后续纵向拖动仍可进入排序。
- 优化边缘胶囊排序的落点判断：拖动释放时按最近槽位重排，横向移开不会被误当作点击唤出。
- 修复边缘激发态胶囊右键菜单的主题资源和标题同步问题，避免独立边缘窗口里的菜单显示旧标题或旧主题。
- 修复隐藏全部纸片时边缘槽位和“收起全部”状态可能残留的问题。
- 修复关闭“胶囊自动贴边”或胶囊模式后，原本停靠在边缘的纸片可能没有恢复出可见窗口的问题。

**胶囊细节**

- 调整普通胶囊和贴边胶囊的关闭区宽度、关闭图标偏移和文字右侧留白，减少白边、错位和误触。
- 贴边胶囊点击判断更稳，普通点击不再容易被误判为拖动。
- 收起全部状态下，边缘胶囊的显示 / 隐藏不再短暂闪出残影。

**设置与提示**

- 新增「显示悬停提示」开关，可关闭按钮和操作区的悬浮提示；设置页的说明图标仍然保留扩展说明。
- 胶囊设置补充说明文字，解释贴边、保留边缘胶囊、收起全部等选项的具体行为。
- 胶囊设置会按依赖关系禁用子选项，父功能关闭时不再显示成仍可直接操作的状态。
- 待办关联笔记关闭时，“关联笔记显示名称”会在设置页禁用；悬停提示说明文案也同步说明设置页说明图标不受该开关影响。
- 修正多语言设置说明中的细节表述：待办关联笔记改为说明“关联入口”打开，外部打开明确使用临时文件，中文 MD 解析说明与按钮文案保持一致。
- 设置窗口在小屏或高缩放环境下会限制高度并允许内容滚动，标题栏和关闭按钮保持可用。
- 贴边胶囊和“收起全部”主胶囊的动画统一遵守“启用动画”开关。

**稳定性**

- 二次启动传递 `--show`、`--hide`、`--new-todo` 等命令时改为短时间重试，降低主实例刚启动时命令丢失的概率。
- `--exit` 在没有主实例时会直接保存退出，不再先恢复窗口或在空数据目录里创建默认待办纸。
- 动画辅助函数会自动修正异常的 Transform 结构，避免部分动画在特殊控件结构下静默失效。
- 待办单项删除和“清理已完成”会先更新数据再播放视觉退出，并忽略已过期的旧行回调，避免快速撤销、刷新或重建列表时出现 stale 状态写回。
- 笔记拖到待办项建立关联时，窗口失焦或关闭会主动取消拖拽反馈，避免临时小窗、光标或目标高亮残留。
- 从待办项打开一个已贴边折叠的关联笔记时，会先走正常展开路径，避免旧边缘槽位残留或笔记窗口不可见。
- 从待办项打开关联笔记时，会按待办窗口所在显示器的工作区就近摆放，避免副屏使用时被夹回主屏。
- 待办排序和边缘胶囊排序在丢失鼠标捕获或窗口关闭时会主动取消，避免残留拖动状态或鼠标光标。
- 优化长笔记的 Markdown 渲染性能，围栏代码块状态会在当前编辑器内缓存，减少滚动到文档后部时的重复扫描。
- 外部打开笔记的文件后缀会拒绝可执行和脚本扩展，避免误把临时文本文件当程序启动。
- 状态文件规范化会清理非有限窗口坐标、尺寸和数组里的空元素，避免异常数据导致加载或保存失败。

### v2.0 革新

> 自 v1.6 后未发版，这一版把 v1.7 起的所有改进合并发布。

**笔记与 Markdown**

- Markdown 解析新增「不启用 / 启用 / 增强」三档，默认增强，切换后笔记立刻刷新。
- 支持把 `---`、`***`、`___` 显示成分割线，列表显示成圆点，任务列表保持整齐，原始符号不再残留。
- 笔记右键菜单整理：单独分出复制 / 粘贴 / 全选；多行选中也能逐行加粗、斜体、删除线。
- 笔记顶栏新增 `MD` 按钮，可用系统默认编辑器打开这篇笔记，打开后缀可在设置里自定义。
- 笔记正文支持 `Ctrl + 滚轮` 按 10% 缩放，右下角显示百分比，点一下回到 100%。

**配色与视觉**

- **四套全新配色**：设置里新增「配色方案」，可在暖纸、墨、林、霞之间切换，每套都配好浅色和深色，即时生效、记住选择。
- 链接、次要文字的颜色随配色相称调整，浅色纸面上更干净。
- 圆角、阴影统一规格：纸片像轻轻搭在桌面上，整体更成套。

**胶囊**

- **胶囊自适应宽度**：按标题长短自动伸缩，保留待办 / 笔记图标；窄数字标题不再撑出大片空白。
- **新增「收起全部」主胶囊**：可在设置里打开顶部主胶囊，一键收起 / 展开全部贴边胶囊；收起后显示「N 个」。
- **贴边胶囊像停靠标签**：露出宽度按图标和标题自适应，关闭区藏回边缘，更像真正停靠在屏幕边上。
- **贴边胶囊支持拖动重排**：在右侧停靠区上下拖动即可调整顺序。
- **展开后保留前台标签**：从贴边位点开纸片后，原槽位保留一个可点的标签，点它把纸片带到最前、点 `×` 直接收起。

**标题与设置**

- **可改纸片标题**：默认「待办1 / 笔记1」，窗口、胶囊、菜单、托盘统一使用；标题最大字数可在设置里调整，默认 6。
- **独立设置窗口**：托盘右键新增「设置」，选项按「外观 / 待办与笔记 / 顶栏按钮 / 外部打开 / 胶囊 / 通用」分组。
- **每个设置项带说明**：每行末尾的 `ⓘ` 悬浮即可看到这项具体做什么、开启后什么效果。
- **顶栏按钮可隐藏**：「新建待办」「新建笔记」「外部打开」三个按钮都能单独关掉。
- **待办可关联笔记**：把笔记拖到待办项上即可建立关联，待办项上可直接打开对应笔记。

**其他**

- 置顶的纸片遇到全屏窗口（如全屏游戏）会主动让开，不再挡住。
- 支持用命令行 / 快捷键控制显示、隐藏、新建等操作，方便配到外部工具。
